Bloque 1 · Conditionals

First Conditional

Para situaciones reales o probables en el futuro.

If + presente simple, ... will + verbo base También: will + verbo base ... if + presente simple
1
La parte con if va en presente.
2
El resultado lleva will + verbo base.
!
Nunca pongas will justo después de if.
Ejemplos
If it rains, we will stay home.
She will pass if she studies.
If you cook eggs, I will make coffee.
If Ann doesn't hurry, she will be late.

Bloque 1 · Conditionals

Zero Conditional

Para verdades generales, hechos científicos y rutinas.

If + presente simple, ... presente simple Aquí if significa when (siempre que…)
1
Las dos partes van en presente.
2
No se usa will en ninguna parte.
3
Siempre que pasa A, pasa B. Resultado seguro.
Ejemplos
If you heat ice, it melts.
If I don't sleep, I get tired.
Plants die if they don't get water.
If you mix oil and water, they separate.

Bloque 2 · Present Perfect

Present Perfect

Acciones del pasado conectadas con el presente (sin decir cuándo exactamente).

have / has + participio pasado I / you / we / they → have  ·  he / she / it → has
+
Afirmativo: She has bought a computer.
−
Negativo: haven't / hasn't + participio.
!
El verbo va en participio, no en pasado simple.
Ejemplos
I have thrown away my old tablet.
Wanda has bought a new computer.
They have looked for information.
Luis hasn't called his parents.

Bloque 3 · Past Participles

Participios pasados

Regulares con −ed; irregulares hay que memorizarlos.

Regular: verbo + -ed play → played  ·  look → looked  ·  finish → finished
!
El participio no es el pasado simple. went ≠ gone
i
En Present Perfect siempre usamos la 3.ª columna.
Irregulares más comunes — base · pasado · participio

eatateeaten
seesawseen
writewrotewritten
dodiddone
breakbrokebroken

taketooktaken
forgetforgotforgotten
Practica · como la sección 5

¿Es el participio correcto?

Marca True (T) si la forma dada es el participio pasado, o False (F) si no lo es.

go → went
✗ Falso: el participio es gone.
play → played
✓ Verdadero: verbo regular, played.
forget → forgotten
✓ Verdadero: forget – forgot – forgotten.
write → writen
✗ Falso: se escribe written (doble "t").
eat → eaten
✓ Verdadero: eat – ate – eaten.
see → saw
✗ Falso: "saw" es pasado; el participio es seen.
do → done
✓ Verdadero: do – did – done.
break → broke
✗ Falso: "broke" es pasado; el participio es broken.
